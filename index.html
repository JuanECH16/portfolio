Proyectos

He trabajado en varios proyectos, tanto personales como profesionales, que han permitido adquirir experiencia en el desarrollo de software y la gestión de proyectos.

MathKid

Proyecto de aula (en desarrollo) de una aplicación web diseñada para ayudar a los niños a aprender matemáticas básicas de manera divertida y efectiva. Desarrollada con Angular, HTML, CSS, JavaScript y TypeScript, MathKid utiliza un enfoque educativo y dinámico para garantizar que los niños puedan aprender matemáticas de manera sencilla y divertida.

Ventas de medicamentos

Proyecto de aula de una aplicación web para la gestión de ventas de medicamentos. Desarrollada con Django, Python, HTML, CSS y JavaScript, la aplicación permite a los usuarios gestionar sus ventas y facturación.

Max Game

Proyecto (en desarrollo) de un videojuego de plataformas en 2D desarrollado con Unity. El objetivo del juego es superar obstáculos y enemigos de todo tipo para llegar al final del nivel.
